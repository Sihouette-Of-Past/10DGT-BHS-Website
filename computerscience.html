Evaluation

For CS, I learned about various interesting concepts using the Burnside Digi Website and tasks on Schoology. First, I learned about the different numerical systems, such as the binary and Hexadecimal system, how to convert them, and their significance in terms of CS. Then I learned about ASCII and UTF8, which are how to express letters and symbols in binary(0s and 1s). After that, I learned about error detection using numerous methods like parity for ASCII and algorithms to find the check digit for barcodes. Finally, I learned about many methods to search and sort data, like bubble sort.

This is a slightly challenging but rewarding topic, with lots of future applications.
